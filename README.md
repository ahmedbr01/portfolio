Ce portfolio présente mes projets et compétences en développement web.

### ✨ Fonctionnalités

- Design responsive
- Section projets avec liens GitHub
- Formulaire de contact fonctionnel (Formspree)
- Optimisation SEO
- Analytics intégré
- Déploiement continu avec Netlify

- HTML5
- CSS3 (Flexbox, Grid)
- JavaScript
- Formspree (formulaire)
- Google Analytics
- Netlify (hébergement)
